📊 Meta Ads Setup Guide

  1. Go to: facebook.com/ads/manager
  2. Create Campaign: Select "Lead Generation" or "Messages"
  3. Budget: Start with $5-10/day
  4. Duration: 2 weeks minimum
  5. Placement: Instagram Feed + Stories + Facebook Feed
  6. Use one ad format above (start with #1 or #2)
